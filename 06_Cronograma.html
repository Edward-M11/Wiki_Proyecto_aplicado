06. Cronograma del Proyecto

El proyecto se estructura en una ejecucin intensiva de 4 semanas (1 mes), diseada para entregar resultados rpidos y de alto valor (Quick Wins).

Fecha de Inicio: 10 de Noviembre Fecha de Finalizacin: 10 de Diciembre

Lnea de Tiempo Detallada

Semana 1: Planificacin y Adquisicin (Nov 10 - Nov 17)

  • Nov 10 - 12: Reunin de inicio (Kick-off), definicin de objetivos y alcance. Aprobacin del uso de Datos Abiertos.
  • Nov 13 - 15: Identificacin y descarga de datasets (Cali y Envigado). Configuracin del entorno de trabajo (GitHub).
  • Nov 16 - 17: Revisin preliminar de la calidad de los datos.

Semana 2: Ingeniera de Datos y EDA (Nov 18 - Nov 24)

  • Nov 18 - 20: Desarrollo de scripts de limpieza (Manejo de nulos, estandarizacin).
  • Nov 21 - 23: Ejecucin del Anlisis Exploratorio de Datos (EDA). Generacin de estadsticas descriptivas.
  • Nov 24: Primer reporte de hallazgos (Insights preliminares).

Semana 3: Visualizacin y Desarrollo Web (Nov 25 - Dic 01)

  • Nov 25 - 27: Diseo de mockups del Dashboard. Seleccin de grficos clave.
  • Nov 28 - 30: Implementacin tcnica del Dashboard (Generacin de HTML/Grficos interactivos).
  • Dic 01: Pruebas de visualizacin y correccin de errores.

Semana 4: Documentacin y Cierre (Dic 02 - Dic 10)

  • Dic 02 - 05: Redaccin final de la Wiki (Descripcin, Riesgos, Manuales).
  • Dic 06 - 08: Revisin final de todos los entregables con el Lder del Proyecto.
  • Dic 09: Despliegue final en GitHub Pages.
  • Dic 10: Entrega oficial y presentacin a Stakeholders.

import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.dates as mdates

# Datos del cronograma
data = [
    {"Task": "Reunin de inicio (Kick-off)", "Start": "2025-11-10", "Finish": "2025-11-12", "Phase": "Planificacin"},
    {"Task": "Identificacin y descarga de datasets", "Start": "2025-11-13", "Finish": "2025-11-15", "Phase": "Planificacin"},
    {"Task": "Revisin preliminar de calidad", "Start": "2025-11-16", "Finish": "2025-11-17", "Phase": "Planificacin"},
    {"Task": "Desarrollo de scripts de limpieza", "Start": "2025-11-18", "Finish": "2025-11-20", "Phase": "Ingeniera de Datos"},
    {"Task": "Ejecucin del EDA", "Start": "2025-11-21", "Finish": "2025-11-23", "Phase": "Ingeniera de Datos"},
    {"Task": "Primer reporte de hallazgos", "Start": "2025-11-24", "Finish": "2025-11-24", "Phase": "Ingeniera de Datos"},
    {"Task": "Diseo de mockups", "Start": "2025-11-25", "Finish": "2025-11-27", "Phase": "Visualizacin"},
    {"Task": "Implementacin tcnica Dashboard", "Start": "2025-11-28", "Finish": "2025-11-30", "Phase": "Visualizacin"},
    {"Task": "Pruebas de visualizacin", "Start": "2025-12-01", "Finish": "2025-12-01", "Phase": "Visualizacin"},
    {"Task": "Redaccin final de la Wiki", "Start": "2025-12-02", "Finish": "2025-12-05", "Phase": "Documentacin"},
    {"Task": "Revisin final", "Start": "2025-12-06", "Finish": "2025-12-08", "Phase": "Documentacin"},
    {"Task": "Despliegue en GitHub Pages", "Start": "2025-12-09", "Finish": "2025-12-09", "Phase": "Documentacin"},
    {"Task": "Entrega oficial", "Start": "2025-12-10", "Finish": "2025-12-10", "Phase": "Documentacin"}
]

df = pd.DataFrame(data)
df['Start'] = pd.to_datetime(df['Start'])
df['Finish'] = pd.to_datetime(df['Finish'])
df['Duration'] = df['Finish'] - df['Start'] + pd.Timedelta(days=1)

# Colores por fase
colors = {
    "Planificacin": "#1f77b4",
    "Ingeniera de Datos": "#ff7f0e",
    "Visualizacin": "#2ca02c",
    "Documentacin": "#d62728"
}

fig, ax = plt.subplots(figsize=(14, 8))

# Invertir orden para que la primera tarea est arriba
df = df.iloc[::-1]

# Crear barras
for i, task in df.iterrows():
    start = mdates.date2num(task['Start'])
    end = mdates.date2num(task['Finish'] + pd.Timedelta(days=1))
    duration = end - start
    
    ax.barh(task['Task'], duration, left=start, color=colors[task['Phase']], edgecolor='black', alpha=0.8)

# Formato del eje X
ax.xaxis.set_major_locator(mdates.DayLocator(interval=3))
ax.xaxis.set_major_formatter(mdates.DateFormatter('%b %d'))
plt.xticks(rotation=45)

# Etiquetas y Ttulo
plt.xlabel("Fecha")
plt.title("Cronograma del Proyecto - Diagrama de Gantt")
plt.grid(axis='x', linestyle='--', alpha=0.7)

# Leyenda
handles = [plt.Rectangle((0,0),1,1, color=colors[label]) for label in colors]
plt.legend(handles, colors.keys(), title="Fases")

plt.tight_layout()
plt.show()
